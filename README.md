#htsw_mvn

前端框架：Bootstrap+FreeMarker
后台框架：EasyUI（包含JS）+Ueditor（文章编辑处理）+Bootstrap（部分美工）+AmazeUI（部分美工）+FreeMarker
    其中：
            涉及到后台整体窗口布局：仅使用EasyUI
            涉及到DataGrid数据处理，前后端数据交互（不包括文章部分）：使用EasyUI数据交互、Bootstrap页面布局优化
            涉及到文章编辑处理：使用Bootstrap页面布局、UEditor文章处理、AmazeUI少量交互优化
服务端框架：JFinal + Tomcat + JDK 1.8 + MySQL
用户权限过滤：shiro + ehcache
中间件：c3p0、java mail、shiro

代码托管：os@git
jar包管理：Maven 3
